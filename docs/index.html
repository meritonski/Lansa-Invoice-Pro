Vanliga frågor (FAQ)

Hur når jag support?

Skicka ett mail via knappen 'Kontakta support' och bifoga gärna skärmdumpar.

Var hittar jag appversionen?

Öppna Inställningar → Om. Version visas högst upp.

Jag ser problem med iCloud/Sync

Kontrollera att iCloud är på för appen i iOS-inställningar. Prova även att starta om appen.

Köp/Pro — återställ köp?

Gå till Profil → Återställ. Om det inte hjälper: kontrollera att du är inloggad på rätt Apple-ID.
